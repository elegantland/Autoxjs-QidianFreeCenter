# Autoxjs-QidianFreeCenter
**起点福利中心自动化脚本 (版本: 260618)**

本脚本基于 [qkqpttgf/Autoxjs-QidianFreeCenter](https://github.com/qkqpttgf/Autoxjs-QidianFreeCenter) 开发，并针对 **一加 Ace 5 至尊版** 进行了深度适配与逻辑增强。

### 🌟 核心优化功能
- **限时福利追踪**：自动识别并优先处理“限时广告·加点”任务。
- **智能阅读流程**：支持自动跳转书架、识别指定书籍、自动阅读并安全返回福利中心。
- **增强型防息屏**：阅读期间每 8 秒执行一次“往返式”复合滑动，确保系统判定为活跃状态。
- **静默跳转模式**：点击跳转类任务后自动在后台计时，无需手动点击“允许跳转”弹窗，倒计时结束后自动拉回。
- **灵活配置**：脚本顶部提供书名自定义、游戏任务开关等配置项。

### ⚙️ 关键配置说明
如果您使用的不是一加 Ace 5，可以通过修改脚本顶部的变量进行微调：
- `targetBookName`: 设置您书架上已有的书籍名称，用于完成“去阅读”福利。
- `enableGameTask`: 设置为 `false` 可跳过耗时较长的玩游戏任务。
- **坐标微调**：若关闭按钮点不到，请开启手机 `开发者选项` -> `指针位置`，观察坐标并修改脚本开头的 `closeButtonBottom` 相关参数。

### 🛠️ 故障排除
1. **控制台闪烁**：这是正常的。脚本在进行 OCR 识图前会主动隐藏控制台以防遮挡关键文字。
2. **任务卡住**：若遇到无法识别的弹窗，脚本通常会在 25 秒后触发强制返回起点的保底逻辑。
3. **权限提示**：请确保 Autox.js 已获得 `无障碍服务`、`悬浮窗` 和 `截图` 权限。
